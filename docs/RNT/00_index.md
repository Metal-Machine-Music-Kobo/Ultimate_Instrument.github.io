#  関係論的ノイズ論 / Relational Noise Theory (RNT)

ここではRNT（関係論的ノイズ論）について章立てで整理しています。  
RNTとは、ノイズを関係の中で生じる差異と捉え、その差異への応答（即興）と循環（フィードバック）によって生成と更新が進む原理を示す理論です。
