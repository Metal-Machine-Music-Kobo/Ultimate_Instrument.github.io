# UI Experiment｜実験映像アーカイブ

アルティメット・インストゥルメントの多様な可能性を提示するシリーズです。  

本ページでは、GitHub 上に配置した短尺動画の埋め込みと、  
外部サービス（Vimeo／YouTube）へのリンクをあわせてご案内します。
