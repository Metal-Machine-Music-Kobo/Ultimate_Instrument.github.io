# ルール / 手順

## 1. セットアップ
- 初期条件の設定（乱数・準備など）
- 記録方法の確認（オーディオ・映像）

## 2. 実行サイクル
1. 聴取と観測  
2. 一点だけ最小修正（±1/5相当）  
3. 必要に応じて微量追加  
4. 閾値到達時：全体へ同相の押し出し処理
